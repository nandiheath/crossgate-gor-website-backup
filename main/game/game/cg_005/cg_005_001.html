□基礎教學 - 寵物忠誠回補方法 ( 筆者： 魔楓 )


＃以下加忠法，只適用於寵物忠誠低於調教下應有忠誠時使用。

　◇調教忠誠度計算方式
　　(調教等級x1000)/寵物等級

　．小數點捨去

　．若有學習調教，寵物最低忠誠度是60
　　(野生寵除外 及 寵物常死亡時有會低於60)

　．詳細請參考[基礎教學][認識調教功用]。


＃何謂加寵忠？

　以調教4、寵物Lv70為例，寵物應是有60忠的，
　但往往寵物會因死亡、主人的魅力降低，或是寵物本身是新封印的野生寵，
　從而導致應有的忠誠下降，即低於原有的60忠。

　如不盡快把忠誠回復應有的忠誠，
　．寵物NONO的頻率便會增加
　．寵物再陣亡、主人的魅力再降低，忠誠便會立即下降
　．忠誠低於某水平時，寵物便會攻擊主人，甚至會逃走，不會再回來

　因此，當寵物忠誠下降了，應盡早補回。


＃如何加寵忠？

　◇寵物升級
　　當寵物升級時，約忠誠低於原調教下應有的忠誠時，便會增加數點，
　　但對於高級的寵物來說，此加忠法不太適合，因為要升級實在太難。

　◇寵忠藥
　　解某些任務或是開寶箱時，會得到一些寵物忠誠變化的物品，
　　例如漂浮的葉子，雖然是有機會加忠，但同時會有副作用，
　　就是對寵物餵食後，是隨機的加
