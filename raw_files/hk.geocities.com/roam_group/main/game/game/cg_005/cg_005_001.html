□基礎教學 - 寵物忠誠回補方法 ( 筆者： 魔楓 )


＃以下加忠法，只適用於寵物忠誠低於調教下應有忠誠時使用。

　◇調教忠誠度計算方式
　　(調教等級x1000)/寵物等級

　．小數點捨去

　．若有學習調教，寵物最低忠誠度是60
　　(野生寵除外 及 寵物常死亡時有會低於60)

　．詳細請參考[基礎教學][認識調教功用]。


＃何謂加寵忠？

　以調教4、寵物Lv70為例，寵物應是有60忠的，
　但往往寵物會因死亡、主人的魅力降低，或是寵物本身是新封印的野生寵，
　從而導致應有的忠誠下降，即低於原有的60忠。

　如不盡快把忠誠回復應有的忠誠，
　．寵物NONO的頻率便會增加
　．寵物再陣亡、主人的魅力再降低，忠誠便會立即下降
　．忠誠低於某水平時，寵物便會攻擊主人，甚至會逃走，不會再回來

　因此，當寵物忠誠下降了，應盡早補回。


＃如何加寵忠？

　◇寵物升級
　　當寵物升級時，約忠誠低於原調教下應有的忠誠時，便會增加數點，
　　但對於高級的寵物來說，此加忠法不太適合，因為要升級實在太難。

　◇寵忠藥
　　解某些任務或是開寶箱時，會得到一些寵物忠誠變化的物品，
　　例如漂浮的葉子，雖然是有機會加忠，但同時會有副作用，
　　就是對寵物餵食後，是隨機的加／降低忠誠度。

　◇打怪加忠法
　　此方法是最有效的加忠法、適合各等級的寵物，亦是全文最主要的部份。

　　□需準備的事項
　　　ぇ人物必須學習調教，調教等級愈高效果愈佳；沒有調教則不能使用此方法。
　　　え寵物盡量學習連擊、反擊，用來清怪用
　　　ぉ若想加效率，人物本身又有技位空出可用，可以學習護衛(若本身已有單補、單恢則不用。)
　　　お將人物的魅力補回到100

　　□忠誠訓練開始
　　　ぇ帶您的寵物到多怪、低Lv的地方打怪(推薦哥布林的家)
　　　え由寵物清怪，為求效率，可使用連擊、反擊等寵技清怪
　　　ぉ人物若有護衛、單補、單恢，每場對寵物使用１次，技等級不限。
　　　お聽說寵物若打飛怪物，比起打死怪物的效果更佳。
　　　か不停的給寵物清怪，加上人物每場替寵謢、補，忠誠慢慢便會增加。

　　□效用解說
　　　ぇ調教愈高Lv，寵物需要打飛、打死的怪物量便會減少。
　　　え經過測試，人物魅力100時，加忠速度會較快。
　　　ぉ每場對寵物使用１次護衛、單補、單恢，另外可+0.1忠。(只限每場１次)
　　　お以調教4、魅100為例，寵物大約飛掉20隻怪便會加１忠，
　　　　即是平均飛１隻怪+0.05忠，而每場使用護、補、恢另+0.1忠。

　　□感想
　　　也許要打飛數十隻怪才有１點忠也要一段時間，但其實也不太難。
　　　哥家一場平均出5隻怪，又少血易死，對一般寵物來說也不太難應付，
　　　而且哥家的出石率不錯，雖然$48粒，也可以空了身上的物品位，
　　　一邊打石一邊練寵忠，賺點外快，一舉兩得。


＃注意事項＃

　街邊不時有玩家說代加寵忠，這些通常也是騙人的，
　99.9%也是藉機會騙走玩家的寵物，所以各位要小心別上當！

　其實要加寵忠根本不用給寵物別人代加，
　只要您有時間，可以解任、開寶箱找寵忠藥，或是用魔幣跟別的玩家買；
　只要您有學調教，就可用打怪加忠法來加忠誠。
　雖然時間是會耐一點，但您寧願付出時間來加忠，還是要一隻會打主人的NONO寵呢？

　最後，以上方法均適合初封印回來的野生寵使用，希望可以幫到大家^__^
